# Algoritmo Genético para Aproximar a Função Ackley

O algoritmo implementa um algoritmo genético com uma representação binária para aproximar o resultado mínimo da função Ackley.

## Compilação e Execução
Exemplo de compilação e execução do código em linguagem C:
```
Make ag
./ag
```

## Entrada e Saída
O programa não possui entrada, portanto os parâmetros para a execução do algoritmo genético são todos definidos no código da atividade.

A saída do programa é feita na saída padrão e é composta por 2 linhas de resultados, contendo o valor mínimo encontrado e os cromossomos do indivíduo.

### Exemplo de Saída do Programa
```
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
                  Resumo dos Resultados
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
 Menor valor: 0.0000009537
 Cromossomos: 1000000000000000000000001010100 1000000000000000000000010101100
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
```
