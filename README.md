# Algoritmos Bioinspirados

Este repositório contém as atividades práticas desenvolvidas durante a disciplina de **Algoritmos Bioinspirados**. O objetivo principal foi implementar e analisar meta-heurísticas inspiradas na natureza para resolver problemas complexos de otimização e busca.

## Estrutura do Repositório 

O projeto está dividido em pastas, cada uma contendo a implementação de um algoritmo específico, aplicado a um problema clássico, e um README detalhando cada uma das atividades, instruções de compilação/execução e entradas/saídas esperadas:

| Pasta / Atividade | Algoritmo | Problema / Função Alvo |
| :--- | :--- | :--- |
| 📂 [**`AckleyPSO`**](./AckleyPSO) | Otimização por Enxame de Partículas (PSO) | Minimização da Função Ackley |
| 📂 [**`AcoPCV`**](./AcoPCV) | Otimização por Colônia de Formigas (ACO) | Problema do Caixeiro Viajante (PCV) |
| 📂 [**`AgBinarioAckley`**](./AgBinarioAckley) | Algoritmo Genético (AG) Binário | Minimização da Função Ackley |
| 📂 [**`AgKnapsack`**](./AgKnapsack) | Algoritmo Genético (AG) | Problema da Mochila (*Knapsack Problem*) |
| 📂 [**`AgPCV`**](./AgPCV) | Algoritmo Genético (AG) | Problema do Caixeiro Viajante (PCV) |
| 📂 [**`AgRealAckley`**](./AgRealAckley) | Algoritmo Genético (AG) Real | Minimização da Função Ackley |
| 📂 [**`ClonalgPCV`**](./ClonalgPCV) | Algoritmo de Seleção Clonal (CLONALG) | Problema do Caixeiro Viajante (PCV) |
